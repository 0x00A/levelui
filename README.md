# DEV
You need a build of leveldown suitable for atom-shell.

```
env HOME=~/.atom-shell-gyp node-gyp rebuild --target=0.19.4 --arch=ia64 --dist-url=https://gh-contractor-zcbenz.s3.amazonaws.com/atom-shell/dist
```
